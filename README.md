
# Slack Chat

Slack Chat is an Atom package that integrates the slack messaging client into the atom text editor.

# Installation
- Navigate to [https://api.slack.com/web](https://api.slack.com/web).
- Under **Authentication**, create a token for your team.
- Go to the slack-chat settings in atom
  - Paste in your token
- Restart atom
